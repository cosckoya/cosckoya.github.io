# DevOps Tools

Essential tools and platforms for modern DevOps workflows - from development to deployment.

## Overview

This section covers practical DevOps tools used in real-world software development, automation, and deployment
pipelines. Each page provides learning resources and official documentation rather than lengthy tutorials.

### Tool Categories

**Version Control & Collaboration**

- [GitHub](github.md) - Code hosting, collaboration, and CI/CD platform
- [Azure DevOps](azure-devops.md) - Microsoft's complete DevOps platform

**Development Tools**

- [Claude Code](claude-code.md) - AI-powered coding assistant CLI
- [Pre-Commit](pre-commit.md) - Git hooks framework for code quality

**CI/CD & Automation**

- [GitHub Actions](github-actions.md) - Workflow automation on GitHub

## Getting Started

**New to DevOps?**

1. Start with **version control** ([GitHub](github.md))
1. Set up **code quality checks** ([Pre-Commit](pre-commit.md))
1. Learn **CI/CD basics** ([GitHub Actions](github-actions.md))
1. Explore **AI-assisted development** ([Claude Code](claude-code.md))

**For Teams:**

- Enterprise platform: [Azure DevOps](azure-devops.md)
- Open source: [GitHub](github.md) + [GitHub Actions](github-actions.md)
