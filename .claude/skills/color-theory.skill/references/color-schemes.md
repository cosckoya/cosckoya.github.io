# Color Schemes & Harmonies

## Classic Color Harmonies

### Monochromatic
**Definition**: Single hue with variations in saturation and lightness

**How to Create**:
- Choose one base hue (e.g., blue at 210°)
- Vary lightness: 20%, 40%, 60%, 80%
- Vary saturation: 30%, 60%, 90%

**Characteristics**:
- Harmonious, peaceful, cohesive
- Easy on eyes, professional
- Can lack contrast/excitement

**Best For**:
- Minimalist designs
- Professional presentations
- Calming interfaces
- Establishing brand consistency

**Example**:
```
Base: Blue (#2196F3)
Light: #E3F2FD (10% saturation, 95% lightness)
Medium: #64B5F6 (60% saturation, 70% lightness)
Dark: #1565C0 (100% saturation, 40% lightness)
Darkest: #0D47A1 (100% saturation, 25% lightness)
```

**Common Mistakes**:
- Not enough contrast for accessibility
- Too many shades making it boring
- Forgetting to add neutral accents

---

### Analogous
**Definition**: 3-5 colors adjacent on color wheel (within 30-60°)

**How to Create**:
- Choose base hue (e.g., 210° blue)
- Add neighbors: 180° (cyan), 240° (violet)
- Use one dominant, others as accents

**Characteristics**:
- Harmonious, natural, pleasing
- Low contrast, cohesive
- One color should dominate

**Best For**:
- Nature-inspired designs
- Gradients and transitions
- Serene, harmonious feels
- Websites, illustrations

**Example**:
```
Blue-Green Scheme:
- 180° Cyan: #00BCD4 (dominant)
- 210° Blue: #2196F3 (supporting)
- 240° Violet: #673AB7 (accent)
```

**Tips**:
- Use 60-30-10 rule (60% dominant, 30% supporting, 10% accent)
- Maintain consistent saturation across colors
- Add neutrals for text and backgrounds

---

### Complementary
**Definition**: Two colors directly opposite on color wheel (180° apart)

**How to Create**:
- Choose base hue (e.g., 30° orange)
- Find opposite: (30° + 180°) = 210° blue
- Use one as dominant, other as accent

**Characteristics**:
- High contrast, vibrant, energetic
- Maximum color contrast
- Can be overwhelming if not balanced

**Best For**:
- Making elements pop
- CTAs and important buttons
- Energetic, bold designs
- Sports, entertainment

**Example**:
```
Orange-Blue:
- Orange: #FF9800 (primary)
- Blue: #2196F3 (accent for CTAs)

Red-Green:
- Red: #F44336 (alerts, errors)
- Green: #4CAF50 (success, go)
```

**Best Practices**:
- Use 70-30 or 80-20 ratio (not 50-50)
- Desaturate one color to reduce vibration
- Add neutrals to calm the scheme
- Avoid red-green for critical info (color blindness)

**Common Mistakes**:
- Equal amounts of both (too intense)
- Both at 100% saturation (vibrating effect)
- Using for large areas (too aggressive)

---

### Split-Complementary
**Definition**: Base color + two colors adjacent to its complement

**How to Create**:
- Choose base (e.g., 210° blue)
- Find complement: 30° orange
- Use neighbors: 0° red, 60° yellow
- Result: Blue + Red + Yellow

**Characteristics**:
- Softer than complementary
- Still high contrast
- More sophisticated, nuanced

**Best For**:
- Beginner-friendly high contrast
- Visual interest without aggression
- Diverse color needs
- Illustrations, infographics

**Example**:
```
Blue + Red-Orange + Yellow-Orange:
- Blue: #2196F3 (dominant)
- Red-Orange: #FF5722 (accent 1)
- Yellow: #FFC107 (accent 2)
```

---

### Triadic
**Definition**: Three colors evenly spaced (120° apart) on color wheel

**How to Create**:
- Choose base (e.g., 0° red)
- Add 120°: 120° green
- Add 240°: 240° blue
- Result: RGB primary triad

**Characteristics**:
- Vibrant, balanced, dynamic
- High contrast, energetic
- Playful, bold

**Best For**:
- Colorful, energetic brands
- Children's products
- Playful, creative designs
- Diverse content needs

**Example - Primary Triad**:
```
- Red: #F44336 (0°)
- Green: #4CAF50 (120°)
- Blue: #2196F3 (240°)
```

**Example - Secondary Triad**:
```
- Orange: #FF9800 (30°)
- Purple: #9C27B0 (270°)
- Teal: #009688 (150°)
```

**Tips**:
- Let one color dominate
- Use others as accents
- Reduce saturation for sophistication
- Works best with unequal distribution

---

### Tetradic (Double Complementary)
**Definition**: Two pairs of complementary colors (rectangle on wheel)

**How to Create**:
- Choose two complementary pairs
- Example: Red-Green + Blue-Orange
- Or: Square (90° apart): Red, Yellow, Green, Blue

**Characteristics**:
- Rich, diverse palette
- Difficult to balance
- Offers most variety

**Best For**:
- Complex designs needing variety
- Infographics with many categories
- Colorful illustrations
- Advanced designers

**Example - Rectangle**:
```
- Red: #F44336 (0°)
- Green: #4CAF50 (120°)
- Blue: #2196F3 (210°)
- Orange: #FF9800 (30°)
```

**Example - Square**:
```
- Red: #F44336 (0°)
- Yellow-Green: #8BC34A (90°)
- Cyan: #00BCD4 (180°)
- Purple: #9C27B0 (270°)
```

**Best Practices**:
- Choose one dominant color
- Use others sparingly
- Consider desaturating some
- Requires careful balance

---

### Accented Neutral
**Definition**: Mostly neutral (gray, beige, white) with one vibrant accent

**How to Create**:
- Base: Gray (#9E9E9E) or Beige (#D7CCC8)
- Neutrals: White, Light Gray, Dark Gray
- Accent: One vibrant hue (any color)

**Characteristics**:
- Sophisticated, modern, clean
- Easy to read and navigate
- Accent creates focal points

**Best For**:
- Minimalist designs
- Professional/corporate sites
- E-commerce (product focus)
- Modern apps and interfaces

**Example**:
```
Neutrals:
- White: #FFFFFF
- Light Gray: #F5F5F5
- Medium Gray: #9E9E9E
- Dark Gray: #424242
- Black: #212121

Accent:
- Teal: #009688 (for buttons, links, highlights)
```

## Modern Color Scheme Patterns

### Duotone
**Definition**: Two colors, often high contrast

**Use**: Overlays, photos, graphic designs
**Example**: 
- Spotify (Black + Bright Green)
- Instagram (Purple-Pink gradient + Orange)

### Gradient Schemes
**Definition**: Smooth transition between 2-3 colors

**Types**:
- **Linear**: Straight line transition
- **Radial**: From center outward
- **Angular**: Sweeping angle

**Best Practices**:
- Use analogous or monochromatic for smooth gradients
- Limit to 2-3 colors
- Ensure text contrast on gradients

**Example**:
```
Instagram-style:
- Purple #833AB4 → Pink #FD1D1D → Orange #F77737

Modern Blue:
- Navy #1A237E → Blue #2196F3 → Cyan #00BCD4
```

### Material Design Color System
**Definition**: Google's color system with primary, secondary, and variants

**Structure**:
- **Primary**: Main brand color + light/dark variants
- **Secondary**: Complementary accent + variants
- **Error**: Red (for errors)
- **Background/Surface**: Neutrals
- **On-Colors**: Text colors that work on each background

**Example**:
```
Primary:
- Main: #6200EE (purple)
- Light: #BB86FC
- Dark: #3700B3

Secondary:
- Main: #03DAC6 (teal)
- Light: #66FFF9
- Dark: #00A896

Error: #B00020
Background: #FFFFFF
Surface: #FFFFFF
```

## Color Palette Sizing

### Minimal (3-5 colors)
```
- Primary color
- Accent color
- Neutral (gray/white)
- Optional: Secondary accent
```
**Best for**: Simple sites, startups, MVPs

### Standard (6-10 colors)
```
- Primary + variants (light, dark)
- Secondary + variants
- Accent
- Neutrals (2-3 grays)
- Success/Error colors
```
**Best for**: Most websites and apps

### Comprehensive (15-20 colors)
```
- Primary palette (5 shades)
- Secondary palette (5 shades)
- Accent colors (2-3)
- Neutrals (5-6 grays)
- Semantic colors (success, warning, error, info)
```
**Best for**: Design systems, large applications

### Enterprise Design System (50+ colors)
- Multiple primary variants
- Full gray scale (10+ shades)
- Multiple accent palettes
- Full semantic color system
- Data visualization palette

**Best for**: Large companies, design systems (IBM, Shopify, etc.)

## The 60-30-10 Rule

**Principle**: Color distribution for balanced designs

- **60%**: Dominant color (usually neutral or brand primary)
- **30%**: Secondary color (supports dominant)
- **10%**: Accent color (creates interest, CTAs)

**Example Website**:
```
60% - White/Light Gray (background, most areas)
30% - Navy Blue (headers, sections, large elements)
10% - Orange (buttons, links, highlights)
```

**Application**:
- **Interior Design**: 60% walls, 30% furniture, 10% accents
- **Web Design**: 60% backgrounds, 30% content areas, 10% CTAs
- **Presentations**: 60% background, 30% main content, 10% emphasis

## Color Scheme Selection Guide

**Choose based on goal:**

| Goal | Recommended Scheme |
|------|-------------------|
| Professional, trustworthy | Monochromatic blue or analogous cool |
| Energetic, attention-grabbing | Complementary or triadic |
| Calming, natural | Analogous green or accented neutral |
| Luxurious, sophisticated | Monochromatic dark + metallic accent |
| Playful, creative | Triadic or split-complementary |
| Minimalist, modern | Accented neutral or monochromatic |
| Bold, impactful | Complementary with high saturation |
| Diverse, varied | Tetradic (advanced) |

## Testing Color Schemes

**Accessibility Check**:
1. Test contrast ratios (WebAIM contrast checker)
2. Simulate color blindness (Stark plugin, Color Oracle)
3. Test in grayscale

**Context Testing**:
1. Different screen sizes
2. Light/dark modes
3. Various lighting conditions
4. Print vs. digital

**User Testing**:
1. Does it match brand personality?
2. Is the hierarchy clear?
3. Are CTAs obvious?
4. Does it evoke intended emotions?
